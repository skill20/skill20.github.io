利用Hexo搭建个人Blog

发表于 2017-01-20

搭建步骤(3步)：

  • 本地环境搭建
  • Github部署
  • Hexo与Github关联

本地环境搭建

  • 安装Git

    • 下载地址

      https://www.git-scm.com/download/win

    • 配置GIT_HOME环境变量，在任意目录打开cmd，验证是否配置成功。

  • 安装Node.js

    • 32位下载地址

      https://nodejs.org/dist/v4.2.3/node-v4.2.3-x86.msi

    • 64位下载地址

      https://nodejs.org/dist/v4.2.3/node-v4.2.3-x64.msi

    • 配置NODE_JS_HOME环境变量，在任意目录打开cmd，输入命令npm install -g hexo验证是否配置成功。

  • 在任意目录下创建一个文件夹，把他命名为hexo，在改目录下打开cmd，输入以下命令：hexo init

  • 在打开的cmd窗口中输入hexo new "your_page_name"，新建文章。

    新建的文章，可以在根目录的source/_posts目录下找到，并且可以使用支持Markdown语法的编辑器来对它进行编辑。

  • 在打开的cmd窗口中输入hexo s --dedug，可以在浏览器查看Blog的预览效果。
